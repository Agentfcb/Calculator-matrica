# Calculator-matrica

#Калькулятор матриц 2x2/3х3
Простая программа на Go для работы с матрицами 2x2/3х3

Возможности
1) Сложение двух матриц

2) Умножение матрицы на число

3) Умножение двух матриц

Как использовать
Установите Go с официального сайта: https://golang.org/dl/

Скачайте файл main1.go с кодом программы

Запустите в терминале:

bash
go run main1.go
Выберите операцию:

Введите 1 для сложения матриц

Введите 2 для умножения матрицы на число

Введите 3 для умножения матриц

Вводите матрицы построчно. Например:

text
1 2
3 4
Пример
text
